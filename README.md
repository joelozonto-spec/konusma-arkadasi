# Konuşma Arkadaşım

Konuşamayan veya konuşma güçlüğü çeken bireyler için tasarlanmış, **kişi/kategori/kelime yönetimi** ve **Türkçe sesli okuma** özellikli web uygulaması.

## Özellikler

✅ **Kişi Yönetimi** - Danışanları isimle kaydedin  
✅ **Kategori Yönetimi** - Her kişiye özel kategoriler oluşturun  
✅ **Kelime/Cümle Yönetimi** - Kategorilere kelime ve cümleler ekleyin  
✅ **Resim Desteği** - Kelime/cümlelere resim ekleyebilirsiniz  
✅ **Sesli Okuma (TTS)** - Türkçe olarak sesli okur  
✅ **PWA Desteği** - Ana ekrana ekleyebilirsiniz  
✅ **Çevrimdışı Mod** - Veriler tarayıcıda kaydedilir  
✅ **Responsive Tasarım** - Mobil, tablet ve desktop'ta çalışır  

## Nasıl Kullanılır

1. **Kişi Ekle**: Sol panelde "Yeni Kişi" butonuna tıklayın
2. **Kategori Seç/Ekle**: Orta panelde kategorileri yönetin
3. **Kelime Ekle**: Sağ panelde "Yeni Kelime/Cümle" butonuna tıklayın
4. **Sesli Okuma**: Herhangi bir kelimeye tıklayıp "Oku" butonuna basın
